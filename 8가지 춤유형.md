## 상세 설명

### 1. 파워풀 ⚔️

| 항목 | 내용 |
|------|------|
| **별명** | 무대 장악형 |
| **핵심** | 강한 힘으로 표현한다 |
| **설명** | 동작 하나하나에 폭발적인 힘이 담긴 당신. 무대 위에서 존재감으로 공간을 지배합니다. |
| **특성** | 파워풀, 역동적, 카리스마, 강약 조절 |
| **한국무용** | 검무, 북춤, 태평무 발딛음 |
| **바이럴 카피** | "네가 추면 무대가 흔들린다" |
| **컬러** | #DC143C (크림슨) |

### 2. 정밀 🌊

| 항목 | 내용 |
|------|------|
| **별명** | 디테일 집착형 |
| **핵심** | 손끝까지 감성을 담는다 |
| **설명** | 손끝 하나도 허투루 두지 않는 당신. 남들이 못 보는 미세한 디테일에 감성을 담아 완벽을 추구합니다. |
| **특성** | 감성, 완벽주의, 장인정신, 디테일 |
| **한국무용** | 살풀이, 감기동작, 뿌림사위, 손목 감기 |
| **바이럴 카피** | "손끝 하나에 혼을 담는다" |
| **컬러** | #4169E1 (로열블루) |

### 3. 미니멀 🌙

| 항목 | 내용 |
|------|------|
| **별명** | 프레임 장인 |
| **핵심** | 동작을 쪼개어 분절한다 |
| **설명** | 하나의 동작을 수십 개의 프레임으로 쪼개는 당신. 분절된 움직임들이 모여 독특한 리듬과 질감을 만들어냅니다. |

| **컬러** | #2F4F4F (다크슬레이트) |
